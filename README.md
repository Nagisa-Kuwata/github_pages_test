# github_pages_test

GitHub Pages の動作確認用リポジトリです。

## 公開 URL

https://nagisa-kuwata.github.io/github_pages_test/

## ファイル構成

```
.
├── index.html   # メインページ
└── style.css    # スタイルシート
```

## GitHub Pages の有効化手順

1. リポジトリの **Settings** を開く
2. 左メニューから **Pages** を選択
3. **Source** を `Deploy from a branch` に設定
4. ブランチを `main`、フォルダを `/ (root)` に設定して **Save**
5. しばらく待つと上記 URL でページが公開される
